What is Binary?
Read together — concept 1 of 3
Base‑2
We normally count in base-10 (denary), using the digits 0 through 9. Binary only uses two digits: 0 and 1.
Every number you can think of can be written using just those two digits.
Read together — concept 2 of 3
What is a Bit?
A bit is a single binary digit — either a 0 or a 1. It is the smallest unit of data in computing.
The word "bit" comes from binary digit.
Read together — concept 3 of 3
Why Binary?
Computers run on electrical signals: ON or OFF. Binary maps directly to this hardware reality — 1 means ON, 0 means OFF.
Every piece of data — text, images, sound, video — is ultimately stored and processed as binary.
What is a Byte?
Definition — byte
A byte is a collection of 8 bits. We group bits together into bytes rather than dealing with them individually. 8 happened to be a convenient number because it could store all English characters.

Binary Place Values
In denary, each column is worth 10× more than the one to its right (1, 10, 100…).
In binary, each column is worth 2× more.
