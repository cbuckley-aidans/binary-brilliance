Binary

Simple Definition

A number system that only uses two digits: 0 and 1.

Detailed Definition

Binary is a base-2 number system used by computers to store and process all types of information. Instead of using ten digits (0-9) like our regular decimal system, binary only uses two digits: 0 and 1. These two digits are called "bits" and they represent the two possible states in electronic circuits: off (0) or on (1).

Why It's Important

  • Binary is the fundamental language used by all computers and digital devices
  • It's perfect for electronic circuits that can have two states: on or off
  • All digital information (text, images, audio, video) is ultimately stored as binary data

Example in Use:

"The number 9 in decimal is written as 1001 in binary, using only the digits 0 and 1."

Bit

Simple Definition

A single binary digit (0 or 1), which is the smallest unit of data in computing.

Detailed Definition

The term "bit" comes from "binary digit" and represents a single value of either 0 or 1 in the binary number system. Bits are the basic building blocks of all digital information. In physical computer hardware, a bit is typically stored by an electronic component that can be in one of two states: "on" (representing 1) or "off" (representing 0).

Why It's Important

  • Bits are the foundation of all digital computing and data storage
  • By combining multiple bits, computers can represent complex information like numbers, text, images, and sounds
  • Bit counts (like 8-bit, 16-bit, 32-bit) are used to measure data capacity and processing power

Example in Use:

"Each character in ASCII encoding is represented by 8 bits of data."

Byte

Simple Definition

A group of 8 bits that can represent 256 different values (from 0 to 255).

Detailed Definition

A byte is a unit of digital information that consists of 8 bits grouped together. Since each bit can be either 0 or 1, a byte can represent 2^8 = 256 different values (0-255). Bytes are often used to represent a single character of text in many encoding systems. They are also the basic unit of measurement for computer memory and storage capacity.

Why It's Important

  • Bytes are used to measure computer memory and storage (e.g., kilobytes, megabytes, gigabytes)
  • In many text encoding systems, one byte represents one character
  • Bytes are a convenient size for computers to process and store information

Example in Use:

"A typical text document might be around 10 kilobytes in size, which is about 10,000 bytes or 80,000 bits of information."

Pixel

Simple Definition

The smallest unit (a tiny coloured square) of a digital image or display.

Detailed Definition

A pixel (short for "picture element") is the smallest controllable element of a digital image or display screen. Images on screens are made up of thousands or millions of these tiny squares arranged in a grid. Each pixel can be set to a specific colour, and together they form the complete image. Pixels are usually so small that you can't see individual ones without zooming in.

Why It's Important

  • Pixels allow computers to display images on screens in a way that digital systems can process
  • The number of pixels (resolution) determines the level of detail in digital images
  • Each pixel's colour information is stored as binary data, with the number of bits determining the range of possible colours

Example in Use:

"If you zoom in close enough on any digital photo, you can see the individual pixels that make up the image."

Colour Depth

Simple Definition

The number of bits used to represent the colour of each pixel in a digital image.

Detailed Definition

Colour depth refers to how many bits are used to store colour information for each pixel in a digital image. More bits means more possible colours. For example, 1-bit colour can only show black and white (2 colours), 8-bit colour can show 256 different colours, and 24-bit "true colour" can display about 16.7 million colours. Deeper colour depth allows for smoother gradients and more realistic images.

How Colour Depth Works

In most modern colour images, the colour depth is divided across three channels:

  • 8 bits for Red (256 levels of red)
  • 8 bits for Green (256 levels of green)
  • 8 bits for Blue (256 levels of blue)

Sampling Rate

Simple Definition

The number of audio samples taken per second when converting sound to digital data.

Detailed Definition

Sampling rate measures how many times per second a sound wave is measured (sampled) when converting analog audio to digital. It is measured in Hertz (Hz) or kilohertz (kHz). For example, CD-quality audio has a sampling rate of 44,100 Hz, which means it takes 44,100 measurements of the sound wave every second. Higher sampling rates can capture higher frequencies and produce more accurate digital representations of sound.

Common Sampling Rates

  • 8,000 Hz (8 kHz): Used for telephone calls and speech-only audio
  • 44,100 Hz (44.1 kHz): CD-quality audio, good for music
  • 48,000 Hz (48 kHz): Standard for professional audio and video production
  • 96,000 Hz (96 kHz) or higher: High-resolution audio for professional studios

Example in Use:

"Professional audio engineers often record at a sampling rate of 96 kHz to capture more detail, even though most listeners can't hear frequencies above what 44.1 kHz can reproduce."

Bit Depth (Audio)

Simple Definition

The number of bits used to represent each audio sample, determining how precisely sound levels can be measured.

Detailed Definition

In digital audio, bit depth measures how many bits are used to store each audio sample. It determines how precisely the amplitude (loudness) of a sound wave can be measured at each sampling point. Higher bit depth allows for more volume levels and better dynamic range. For example, 16-bit audio (CD quality) can represent 65,536 different volume levels, while 24-bit audio (professional quality) can represent over 16 million levels.

Common Bit Depths

  • 8-bit: 256 volume levels (basic speech, older games)
  • 16-bit: 65,536 volume levels (CD quality audio)
  • 24-bit: 16,777,216 volume levels (professional audio)
  • 32-bit float: Even greater precision (studio production)

Example in Use:

"Professional recording studios typically use 24-bit bit depth because it allows for more precise volume levels and captures quiet sounds better than the 16-bit format used for CDs."

ASCII

Simple Definition

A standard encoding system that assigns numbers to letters, digits and symbols, allowing computers to store and transmit text.

Detailed Definition

ASCII (American Standard Code for Information Interchange) is a character encoding standard created in the 1960s. It assigns a unique number to each letter, digit, punctuation mark, and common symbol. The original ASCII used 7 bits per character, allowing it to represent 128 different characters (0-127). Extended ASCII uses 8 bits (1 byte) for 256 characters. ASCII was one of the first widely-used standards for representing text in computers.

How ASCII Works

  • Each character is assigned a number (e.g., 'A' = 65, 'a' = 97, '0' = 48)
  • This number is converted to binary for computer storage
  • 7-bit ASCII covers basic English letters, numbers, and common symbols
  • 8-bit Extended ASCII adds additional symbols and characters

Example in Use:

"The letter 'A' is represented in ASCII as the decimal number 65, which is stored as the binary number 01000001."

Unicode

Simple Definition

A universal character encoding standard that includes characters from virtually all languages and writing systems in the world.

Detailed Definition

Unicode is an international character encoding standard designed to represent text from all the world's writing systems. Unlike ASCII, which was mainly limited to English characters, Unicode can represent characters from languages such as Chinese, Arabic, Russian, Hindi, and many others. It also includes symbols, emoji, and historical scripts. Unicode assigns a unique number (code point) to each character, regardless of platform, program, or language.

Why It's Important

  • Unicode enables computers to display and process text in any language
  • It allows global communication and information sharing across different writing systems
  • Most modern software and websites use Unicode (often in UTF-8 format) to support international content
  • Unicode can represent over 143,000 characters and continues to grow

Digital Representation

Simple Definition

The process of converting real-world information into binary data that computers can process and store.

Detailed Definition

Digital representation refers to how computers convert and store real-world information (like text, images, sounds, and numbers) as patterns of binary digits (1s and 0s). Different types of data use different methods of digital representation, but all ultimately become binary code. This conversion allows computers to process, store, transmit, and manipulate all kinds of information using the same basic system.

Types of Digital Representation

  • Text: Letters and symbols are converted using encoding standards like ASCII or Unicode
  • Images: Pictures are represented as grids of pixels, each with colour information
  • Audio: Sound waves are sampled at regular intervals to create digital values
  • Video: Combines image representation with timing information
  • Numbers: Converted directly to binary using various numbering systems

Example in Use:

"The digital representation of a colour photograph involves storing information about the colour of each pixel as a binary value."
