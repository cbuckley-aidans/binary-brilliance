CHUNK 1: BINARY BASICS

TEACHER CHUNK
5 mins

How Computers See Text

Computers only understand numbers - specifically, binary numbers made up of 1s and 0s. They can't directly understand letters, symbols, or emoji the way humans do.

1

What is Binary?

Binary is a number system that only uses two digits: 0 and 1.

Each 0 or 1 is called a bit (binary digit).

01000001 = The letter "A"

Binary is the language of computers because electronic components can easily represent two states: ON (1) and OFF (0).

2

Why We Need Text Encoding

For computers to work with text, we need a system that converts between:

Hello!
01001000 01100101 01101100...

This conversion system is called text encoding. It's like having a special code book that tells the computer which binary numbers represent which characters.

3

Bytes: Grouping Bits

A single bit (0 or 1) doesn't give us many options - just two! To represent more information, computers group bits together into bytes.

0 1 0 0 0 0 0 1

1 byte = 8 bits

With 8 bits in a byte, we can represent 256 different values (from 0 to 255). That's enough for all the letters, numbers, and basic symbols we need!

Key Points

COMPUTERS SPEAK BINARY

Computers only understand binary (1s and 0s)

TEXT NEEDS ENCODING

We need a system to convert between text and binary

BYTES STORE CHARACTERS

Each character is typically stored as one byte (8 bits)

Your Binary Finger Alphabet

Just like computers use patterns of 1s and 0s to represent text, you'll create a language using just your fingers!

How it works:

  1. Each finger can be either UP (1) or DOWN (0)
  2. With 10 fingers, you can create 10-bit binary patterns
  3. Different patterns will represent different letters or words
  4. You'll "speak" by showing different finger patterns

ASCII: The Text Encoding System

In the early days of computing, a system called ASCII (American Standard Code for Information Interchange) was created to standardize how text should be represented in binary.

1

How ASCII Works

ASCII assigns a unique number to each character:

  • 'A' is assigned the number 65
  • 'B' is assigned the number 66
  • 'a' is assigned the number 97
  • '1' is assigned the number 49
  • Space is assigned the number 32

These numbers are then converted to binary for the computer to process!

2

ASCII in Binary

Since ASCII uses numbers from 0 to 127, we only need 7 bits to represent all ASCII characters. But computers typically use 8 bits (1 byte) for convenience.

Character ASCII Value Binary
A 65 01000001
B 66 01000010
a 97 01100001
Space 32 00100000
3

Text to Binary Conversion

Let's see how the word "Hi" is converted to binary:

Converting "Hi" to Binary

Step 1: Identify each character

H
i

Step 2: Look up the ASCII value for each character

H
ASCII: 72
i
ASCII: 105

Step 3: Convert each ASCII value to binary

H
ASCII: 72
01001000
i
ASCII: 105
01101001

Final result: "Hi" in binary

01001000 01101001
4

ASCII Table

Here's a portion of the ASCII table showing lowercase letters:

Character ASCII Binary Character ASCII Binary
a 97 01100001 n 110 01101110
b 98 01100010 o 111 01101111
c 99 01100011 p 112 01110000
d 100 01100100 q 113 01110001

Notice a pattern? Lowercase letters start at 97 ('a') and continue in sequence.

TEACHER CHUNK

From ASCII to Unicode: Supporting Global Text

ASCII was great for English, but has a major limitation - it only supports 128 characters! This isn't enough for other languages and symbols.

1

The Limitations of ASCII

ASCII can only represent:

English Letters

A-Z, a-z

Numbers

0-9

Basic Symbols

! @ # $ % ^ & * ( ) etc.

But ASCII cannot represent:

Other Languages

Chinese, Arabic, Russian

Emojis

Special Symbols

2

Unicode to the Rescue

Unicode was created to solve ASCII's limitations. Instead of just 7-8 bits, Unicode uses multiple bytes to represent characters from all languages and systems.

Unicode can support over 1.1 million characters! That's enough for all human languages - even ancient ones.

Chinese Character "N" (You)

Unicode: U+4F60

Binary: 11100100 10111101 10100000

Russian Letter "Zh"

Unicode: U+0416

Binary: 11010000 10010110

Smiling Face Emoji

Unicode: U+1F60A

Binary: 11110000 10011111 10011000 10001010

3

UTF-8: The Most Common Unicode Format

There are different ways to encode Unicode characters in binary. The most common is UTF-8, which is used by most websites and systems today.

UTF-8's Smart Design:

  • Uses 1 byte for basic Latin/ASCII characters (compatible with ASCII)
  • Uses 2-4 bytes for other characters
  • Can represent any Unicode character

Unicode enables digital text in any world language

CHUNK 4: REAL-WORLD APPLICATIONS

TEACHER CHUNK
5 mins

Why Text Encoding Matters

Text encoding might seem like a technical detail, but it affects our digital lives in many ways:

1

Digital Communication

Every text message, email, and social media post you send is encoded as binary when it travels across the internet.

Without standardized text encoding, people using different devices or from different countries couldn't communicate easily!

2

Global Websites

Websites need to display content in many languages. Unicode makes this possible by providing a standard way to encode text from any language.

Most modern websites use UTF-8 (a Unicode encoding) to support international users.

3

Storage and Security

Data Storage

Text encoding affects how much space your text files take up. Different encoding methods can make files larger or smaller.

Security

Many security systems use text encoding for passwords and data encryption to protect your information.

Example: When you create a password online, the website encodes it and stores it in binary form to keep it secure.

4

A Brief History

1960s

ASCII is created, allowing computers to represent 128 characters

1980s

Different countries create their own encoding systems for their languages

1991

Unicode is introduced to unify all character encodings

Today

Unicode supports over 143,000 characters and continues to grow with new emoji and symbols
