CHUNK 2: INTRODUCTION TO BINARY

TEACHER CHUNK

Binary: The Language of Computers

Computers don't understand decimal numbers like we do. Instead, they use a system called binary, which only has two digits: 0 and 1.

1

Why Binary?

Computers use binary because they're built using electronic components that can have two states:

ON

1

OFF

0

These on/off states are represented by electrical signals:

  • ON = High voltage = 1
  • OFF = Low voltage = 0
2

Binary Digits (Bits)

Each 0 or 1 in binary is called a bit (binary digit).

Single Bit

Can only represent two values: 0 or 1

Multiple Bits

Combine to represent larger numbers

Byte (8 bits)

Common grouping used in computers

Fun Fact: One byte (8 bits) can represent numbers from 0 to 255 (a total of 256 different values).

3

Binary Place Values

In our decimal system, each digit position represents a power of 10.
In binary, each position represents a power of 2:

Position 27 26 25 24 23 22 21 20
Value 128 64 32 16 8 4 2 1
Example 1 0 1 0 1 0 0 1

To convert from binary to decimal:

  1. Check each position with a "1" in it
  2. Add up the values of those positions
  3. The sum is your decimal number

Example: 10101001 in binary

128 + 0 + 32 + 0 + 8 + 0 + 0 + 1 = 169

CHUNK 3: BINARY REPRESENTATION

TEACHER CHUNK
15 mins

Converting Between Decimal and Binary

1

Decimal to Binary

To convert from decimal to binary:

  1. Find the largest power of 2 that fits into your number
  2. Put a 1 in that position and subtract that value
  3. Repeat with the remainder
  4. Put 0s in any positions you don't use

Example: Convert 42 to binary

42 - 32 = 10 (use 25)

10 - 8 = 2 (use 23)

2 - 2 = 0 (use 21)

Result: 00101010

2

Binary to Decimal

To convert from binary to decimal:

  1. Identify which positions have a 1
  2. Find the value of each position (powers of 2)
  3. Add all these values together

Example: Convert 10110 to decimal

Position values: 16, 8, 4, 2, 1

Positions with 1s: 16, 4, 2

16 + 4 + 2 = 22
